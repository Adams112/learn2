输入输出
0	标准输入
1	标准输出
2	错误输出
输入
	<	文件
	<<	分界符
	< 文件1 > 文件2
输出
	> 文件	标准输出到文件
	2> 文件	错误输出到文件
	>> 文件	标准输出追加到文件
	2>> 文件	错误输出追加到文件
	>> 文件	2>&1	标准输出和错误输出追加到文件
	&>>				标准输出和错误输出追加到文件
